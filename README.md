# Roblox Server Join Link Generator
A simple program that grabs the last Roblox server you joined and turns it into a join link.

# Usage
Requires .NET 8.0 framework installed! <br>
Extract the .zip file and then open the .exe file, and it should copy the generated join link by default. Then, you can send the generated link for anyone to open, and they'll join to your Roblox server.
